confgraph
=========

WORK IN PROGRESS

A tool to draw dependency graphs for [ddts](https://github.com/maddenp/ddts) configurations.

1. lein uberjar
2. cd [$DDTSAPP](https://github.com/maddenp/ddts/blob/master/README)
3. java -jar /path/to/confgraph-1-standalone.jar [filter]
4. ls graph.svg
